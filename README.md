# T2 industrial apac teams comments report generator

Track T2 member Teams messages across channels/threads for BESS project sites. Uses Microsoft Graph API to pull data with custom date ranges and exports to CSV for performance analysis.

---

## 📌 Features
- Pulls message volume for each T2 member from Teams channels & threads via Microsoft Graph API  
- Flexible date range input (no fixed window)  
- Exports structured CSV for analysis  
- Designed for BESS project site communication tracking  

---

## 🔐 Setup
1. Enter refresh token
2. Enter start/end dates (YYYY-MM-DD) when prompted
